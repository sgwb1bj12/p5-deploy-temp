Practicas de laboratorio

Practica 1

Tema: HTML, CSS y fundamentos web

Introduccion al desarrollo web estatico. Creacion de paginas con HTML semantico y estilos CSS. Conceptos basicos de maquetacion y responsividad.

Practica 2

Tema: JavaScript y comunicacion con el servidor

Programacion del lado del cliente con JavaScript. Validacion de formularios, manipulacion del DOM y comunicacion con una API Flask mediante fetch.

Practica 3

Tema: Flask y Jinja2

Desarrollo de una aplicacion web dinamica con Flask. Uso de plantillas Jinja2 con herencia, rutas, controladores y un servicio de chat del lado del servidor.

Practica 4

Tema: Persistencia y seguridad

Integracion de base de datos con SQLAlchemy. Operaciones CRUD, migraciones con Alembic y aplicacion de medidas de seguridad contra vulnerabilidades comunes.

Normas de entrega

  • Las practicas se entregan a traves de GitHub con tags/releases
  • Se debe mantener la estructura de carpetas indicada en cada enunciado
  • No incluir informacion personal ni identificativa en los archivos
  • Se valorara la organizacion, la calidad del codigo y la comprension demostrada

Criterios de evaluacion

Funcionalidad

La aplicacion debe cumplir todos los requisitos descritos en el enunciado. Se comprobara que las rutas, formularios y servicios funcionan correctamente en distintos navegadores.

Calidad del codigo

Se valora la legibilidad, la organizacion en modulos, el uso correcto de patrones como MVC y la ausencia de codigo duplicado o innecesario.
